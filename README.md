# Python Logic Building

This repository contains beginner-level Python programs created to improve problem-solving and logic-building skills.

## Concepts Covered

### 1. Sum of Numbers
Calculate the sum of numbers from 1 to n.

### 2. Even or Odd
Check whether a number is even or odd.

### 3. Factorial
Find the factorial of a given number.

### 4. Count Digits
Count the number of digits in a number.

### 5. Sum of Digits
Calculate the sum of all digits in a number.

### 6. Reverse Number
Reverse a given number.

### 7. Palindrome Number
Check whether a number is a palindrome.

### 8. Prime Number
Check whether a number is prime.

### 9. Nested Loops
Practice nested loop concepts.

### 10. Star Patterns
Print different star patterns using loops.

### 11. Fibonacci Series
Generate the Fibonacci sequence.

### 12. Armstrong Number
Check whether a number is an Armstrong number.

## Technologies Used

- Python 3

## Purpose

This repository was created to strengthen Python fundamentals and improve logical thinking through coding practice.
